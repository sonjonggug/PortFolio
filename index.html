한약재 판매 웹사이트

2022. 03. 21 ~ 2022. 04. 20 (진행중)

프로젝트 소개

- 우리나라에서는 건강에 대해 한약/한약재를 꾸준히 소비하는 경향이 있어 고정 소비자층 확보 가능

- 한약재를 이용한 식품이나 기능성 제품이 환경호르몬의 영향을 상쇄하고, 면역력을 강화하는 등의 연구결과에 따라 한약재에 대한 관심 또한 증가하는 추세

- 직접적인 한약제조를 하는 한의원에게 판매하는 원물약재와, 구매량이 적거나 환이나 즙과 같은 바로 복용하기 편한 제품을 원하는 일반고객까지 폭넓게 수용할 수 있는 약재판매사이트를 구현.

주요 업무 및 상세 역할

  • Chart.Js와 Ajax, SQL문을 활용하여 일별 월별 년별 제품 판매량과 매출액,판매비율을 통계내어 데이터 시각화 구현
  • 결제된 매출 리스트 조회 및 페이징 구현
  • 판매액 높은 순 , 낮은 순 , 신 상품 순 정렬 및 페이징 구현
  • 조건 별 검색 구현 및 페이징 구현
  • Ajax를 활용한 판매 상품 상세 리스트 및 페이징 구현
  • Session을 활용한 방문자수 체크 및 조회 구현
  • Rest방식으로 Url 정리 및 세션을 활용한 상품 정보 숨김
  • 통합
  • 아마존 호스팅

사용 언어 및 개발 환경

- OS : Windows10, Ubuntu Linux

- Database : MySQL/MariaDB

- Service : Apache Tomcat, AWS

- 언어 : Spring4, MyBatis, Spring Rest API, JAVA, JSP, HTML5/CSS3,jQuery, XML, Ajax, Bootstrap

- Tool : Eclipse(STS)

- 오픈 API : Chart.js

- 형상관리 : Git/Github

- Github : https://github.com/sonjonggug/finalProject.git

렌터카 예약 웹사이트
